# Timer-Pomodoro

Projeto desenvolvido atraves de aulas para aprofundar e firmar os conhecimentos com as tecnologias que foram utilizadas.
que foram: 
- React
- Typescript
- HTML 
- CSS
- Styled Componentes
- React Router DOM
- Hooks (Custom, Context, Reducer, Form)
